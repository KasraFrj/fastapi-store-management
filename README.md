# 🛒 سیستم جامع مدیریت فروشگاه (Comprehensive Store Management System)

این پروژه یک سیستم بک‌اند فروشگاهی است که با فریم‌ورک **FastAPI** توسعه یافته و برای مدیریت محصولات، کاربران و فرآیندهای فروشگاهی طراحی شده است. کل ساختار پروژه با استفاده از **Docker** داکریزه شده و آماده راه‌اندازی سریع است.

## 🛠️ تکنولوژی‌های استفاده شده
- **FastAPI** (پایتون ۳.۱۱)
- **PostgreSQL** (دیتابیس اصلی آسنکرون به همراه `asyncpg`)
- **Redis** (لایه کَش و آماده‌سازی برای سبد خرید)
- **Docker & Docker Compose** (مدیریت کانتینرها)

---

## 🚀 راهنمای راه‌اندازی و اجرا (How to Run)

برای اجرای این پروژه روی سیستم خود، نیازی به نصب پایتون یا پستگرس ندارید! فقط کافیست مراحل زیر را دنبال کنید:

### ۱. شبیه‌سازی پروژه (Clone)
ابتدا پروژه را از گیت‌هاب کلون کنید و وارد پوشه پروژه شوید:
```bash
git clone [https://github.com/KasraFrj/fastapi-store-management.git](https://github.com/KasraFrj/fastapi-store-management.git)
cd fastapi-store-management
