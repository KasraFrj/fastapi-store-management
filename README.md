# 🛒 Comprehensive Store Management System API

این پروژه یک API کامل برای مدیریت فروشگاه اینترنتی است که با استفاده از فریم‌ورک محبوب **FastAPI** و دیتابیس **PostgreSQL** توسعه یافته است.

## 🚀 ویژگی‌های کلیدی پروژه:
- **احراز هویت امن (Authentication):** ثبت‌نام و لاگین کاربر با سیستم هش پسورد (`bcrypt`) و صدور توکن امنیتی `JWT`.
- **مدیریت محصولات (Product CRUD):** سیستم کامل ساخت، نمایش، ویرایش و حذف محصولات در دیتابیس.
- **سبد خرید هوشمند (Cart System):** مدیریت سبد خرید کاربران لاگین شده و اتصال آیتم‌ها به محصولات دیتابیس با استفاده از Relationships در SQLAlchemy.
- **مستندات خودکار (Swagger UI):** داکومنت کامل و تست آنلاین تمام روت‌ها به همراه دکمه Authorize برای تست توکن‌ها.

## 🛠 تکنولوژی‌های استفاده شده:
- FastAPI
- PostgreSQL & SQLAlchemy (ORM)
- PyJWT & Passlib
- Uvicorn
